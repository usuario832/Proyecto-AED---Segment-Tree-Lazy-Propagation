# Proyecto AED: Segment Tree con Lazy Propagation

## PASOS PARA EJECUTAR EL PROGRAMA

### Backend

1. **Ubicación del archivo:** Asegúrate de que el archivo de base de datos llamado `VentasTambo.db` se encuentre en la carpeta raíz del proyecto (donde está el archivo `main.go`).

2. **Descarga y sincroniza las dependencias necesarias:**

    ```bash
    go mod tidy
    ```

3. **Ejecutar:**

   ```bash
   go run .
    ```

### Frontend (En una nueva terminal)

1. **Ir a la ubicación del archivo del FRONTEND:**

    ```bash
    cd interfaz-proyecto
    ```

5. **Instalar Dependencias:**

    ```bash
   npm install
    ```

3. **Ejecutar**

    ```bash
   npm run dev
    ```

4. **Entrar a http://localhost:5173**
